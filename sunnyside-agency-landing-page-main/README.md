## 피드백

- svg fill을 통해 색상을 바꿀 때,
  svd의 path에 attribute로 fill을 쓰던지,
  아니면 css fill을 활용한다.
  path에 fill이 있으면 css fill이 적용되지 않는다.

- background-image를 활용할 때,
  background-size에서 background-image의 width와 height조절가능하다.
