## 피드백

- img element안에 svg사용할 때,
  svg의 색을 바꾸고 싶다면, filter를 활용한다.
  아니면 아예 svg태그를 사용해 놓고 fill을 통해 색을 변화시킨다.
  아이콘 같은 경우에는 stroke를 활용한다.

- pseudo element를 잘 활용해보자

- flex-grow를 주의를 가지고 쓰자.
